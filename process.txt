Here are some learnings
    XDP
        - in kernel space inside NIC
        - no memory alloc or copying packets to userspace
        - can pass, drop, redirect, tx, or abort
    dBPF
        - system to run small verified progams in kernel
    VIP of load balancer
        - can be public or private depending on internal or external facing or priv network 
    AWS
        - Putting services on same subnet and VPC
        - Setting security groups, TCP, SSH, ICMP access
    Data Plane vs Control Plane
    iperf3
    Hash Rings - only flows by new/deleted servers are remapped but uneven distribution
    Rendezvous hashing - good for dozens bad for thousands b/c lookup hashes against every server, good dist
    Maglev hashing - table of backend servers that is 0(1) lookup, optimal update from maglev alg
